== Description ==

Puts Google +1,Google buzz buttons of your choice above or below your posts.
Easy customization of active buttons and position in the Settings menu.

More info: http://emsphere.info/wp-plugin/google-1google-buzz-buttons.html


== Installation ==
1. Upload the `Google +1,Google buzz` directory into the `/wp-content/plugins/` directory
2. Set your favourite values in the `Settings->Google +1,Google buzz buttons` menu in Wordpress
3. Activate the plugin through the `Plugins` menu in WordPress
4. Enjoy!
